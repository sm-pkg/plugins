# smpkg-repo

A repository containing plugins for sourcemod for use with [smpkg](https://github.com/leighmacdonald/smpkg).

## A Package Definition

Example smpkg.yaml

```yaml
version: 0.1.0
name: afkmanager4
authors:
  - Someone
description: |
  A package for managing AFKs in a game.
homepage: https://github.com/username/afkmanager4
license:
dependencies: ["another-package-name"]
```
